Absendo is a tool that automatically fills out absence forms for the BBZW (Berufsbildungszentrum Wirtschaft, Informatik und Technik).

## 🌐 Live Demo

Visit the app here: [https://absendo.app](https://absendo.app)

## 🚀 Technologies

- **TypeScript**
- **Vite**
- **Vercel** (Deployment)
- **Node.js** (Backend)
